Linux Mint 17.2 Upgrade with Virtual Box install

(This documentation is for the electronic students at WCJC)

This is documentation on upgrading a Linux Mint 17.1 system to the Linux Mint 17.2 operating system. This includes the installation of Virtual Box and Windows XP as well as installing programs and drivers to run the Mitsubishi PLC Trainer from the PC.

Installing Linux Mint 17.2

(I apologize for the lack of pictures in the beginning, but the rest of the documentation should have pictures that make it super easy to follow)

1. Reboot your PC. As it boots back up hit F2 to enter the BIOS

2. Use the arrow keys on your keyboard and select the BOOT Tab.

3. Make sure that the DVD drive is first in the Boot Priority List.

4. Use the arrow keys on your keyboard to select the EXIT tab, and select Exit Saving Changes.

5. Allow your PC to boot back up.

6. Once the PC has rebooted, open the DVD drive and insert the Linux Mint 17.2 installation DVD in the drive.

7. Reboot the PC again.

8. As it is rebooting you will notice it is a different boot screen.

9. Once it gets to the Login screen you can either Hit enter to login or after 10 seconds it will auto-login.
